# Data Directory

This directory contains datasets used in the project.

- `raw/` and `processed/` data are generated or downloaded locally and are **not versioned**.
- All datasets in `processed/` are fully reproducible via scripts in `scripts/`.
- The datasets in `raw/` are sourced from publicly available online resources; details and links can be found in the main README.

See the main README for data acquisition and generation instructions.
